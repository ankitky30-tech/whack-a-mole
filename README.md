# 🐹 Whack A Mole - Arcade Edition

A fast-paced, mobile-style Whack-a-Mole game built with HTML, CSS, and JavaScript.  
Designed with arcade “juice”, combo systems, and mobile-first interaction feedback.

---

## 🔗 Live Demo
👉 https://ankitky30-tech.github.io/whack-a-mole

---

## 🎮 Gameplay

- Tap moles as they appear
- Earn points for each hit
- 🔥 Build combo streaks for faster scoring
- 🐹 Rare golden moles give bonus points
- Game gets faster as your level increases
- Survive until the timer ends

---

## ✨ Features

### 🎯 Core Gameplay
- Random mole spawning system
- Score tracking system
- Timer-based rounds (30 seconds)
- Level-based difficulty scaling

### 🔥 Arcade “Juice”
- Combo multiplier system (x1, x2, x3…)
- Golden rare mole system (+bonus points)
- Screen flash on hit
- Shake effect on miss
- Hit animations with scaling feedback

### 📱 Mobile-Style Design
- Touch-friendly interactions
- Vibration feedback (mobile devices)
- Fast response input system
- Minimal UI latency for arcade feel

### 💥 Game Feel Enhancements
- Smooth animations
- Dynamic difficulty curve
- Instant feedback loop (sound + visual + vibration)

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (animations + effects)
- Vanilla JavaScript (game logic)
- Web Audio API (sound effects)
